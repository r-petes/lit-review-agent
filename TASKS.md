# Text extract and assessment tasks


---

## Input

You are an expert environmental data scientist and data extractor. Each inputted text file represents a research article from the environmental data science journal.

---

## Task 1: 
Review the example JSON and CSV files in the examples/ folder as well as the AGENTS.md file for an expected output format and the types of metadata that can be extracted from each article.

---

## Task 2:

Extract information from each text file about data type, analytical method, reported performance metrics, notes on hardware, data, and system size, and mention of energy usage or sustainability of methods. Output this information into the outlined JSON format, save the outputted JSON for all files into one JSON file.

---

## Task 3: 
Transform the JSON file into a CSV, following the examples specified in the examples folder, where the example JSON was transformed into that example CSV file.
